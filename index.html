mess-manager

What?

A peer to peer distributed encrypted storage and message manager.

Why?

  • I trust people in groups I assign more than I trust lobbying corps.
  • I would like to see multi-player games go serverless, adless, community driven, and free.
  • Bypass Censorship, Filtering, and discriminatory business practices.

How?

The manager API provides network messaging, peer management, and file storage to Applications. Under the hood, sqlite databases contain information on peers (and peers of peers) last connection addresses and protocol preferences and file and group tables. crypto++ provides a menu of encryption strategies. Mess Manager follows various strategies for updating, propagating, and encrypting different kinds (Clumps) of data and messages to peers.

  • A Mess is your local storage state.
  • A Clump is a security and management group within the Mess following specified strategies for encryption, propagation to other peers, and update triggers.
  • A Nugget is one of many files in a Clump allowing for fractional or statistical propagation of clumps

Potential implementations:

  • Social network app with embedded game api.
  • Medium sized multiplayer games.
  • Document Collaboration
  • Mutual security apps
  • Backup to friends
  • Distributed simulations
